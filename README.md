# ecstatic-issues
Issues tracker for the minecraft LOD mod Ecstatic. Please refer to the curseforge page at https://www.curseforge.com/minecraft/mc-mods/ecstatic.

# Reporting a Bug
Before opening a new issue, please use the search bar to see if your bug has already been reported.

When you make a bug report, please include 
- **Minecraft Version** (eg. 1.20.1)
- **Modloader and version** (eg. Forge 47.1.3)
- **Mod Version**
- **Description** a clear description of the problem (Photos are extremely helpful)
- **Steps to reproduce** How can we make this bug happen again?
  
Crash report is not required but would be extremely helpful 

# Requesting a feature

If you would like a feature to be added to Ecstatic, please leave it as an issue with a clear description of the feature!
